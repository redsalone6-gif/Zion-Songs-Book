GitHub Pages can host the app screens and navigation. It does not run a database or server-side features. For login, saving songs, or syncing data, you need a backend such as Firebase, Supabase, or Google Apps Script.

Publish on GitHub

  1. Create a new GitHub repository.
  2. Extract this ZIP on your computer.
  3. Upload all extracted files to the repository root.
  4. Open Settings → Pages.
  5. Choose Deploy from a branch, branch main, folder / (root).
  6. Save. Your site will publish at the GitHub Pages URL for that repository.
