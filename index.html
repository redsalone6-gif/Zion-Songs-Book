What is working now

Add song form, song library, detail page, favorites, collections, and browser storage all work as a real frontend app.

What still needs backend

Cross-device sync, login, online database, admin roles, and shared libraries need Firebase, Supabase, or Google Apps Script.

Tuner note

The tuner page includes a real microphone permission test. A full pitch-detection tuner would need additional audio-processing logic.

Deploy this folder directly to GitHub and enable GitHub Pages from the root branch.
